Get-VPASAuthenticationMethods


  • Synopsis
  • Description
  • Syntax
  • Parameters
  • Examples
  • Outputs
Supported Environments:
SelfHosted
SharedServices
PCloudStandard


Synopsis:

GET AUTHENTICATION METHODS


Description:

USE THIS FUNCTION TO GET ALL THE AUTHENTICATION METHODS INTO CYBERARK


Syntax:

Get-VPASAuthenticationMethods [[-token] <Hashtable>] [<CommonParameters>]


Parameters:

-token <Hashtable>
HashTable of data containing various pieces of login information (PVWA, LoginToken, HeaderType, etc).
If -token is not passed, function will use last known hashtable generated by New-VPASToken

Required: false
Position: 1
Default value:
Accept pipeline input: true (ByPropertyName)
Accept wildcard characters: false

<CommonParameters>
This cmdlet supports the common parameters: Verbose, Debug
ErrorAction, ErrorVariable, WarningAction, WarningVariable
OutBuffer, PipelineVariable, and OutVariable. For more information, see
about_CommonParameters (https:/go.microsoft.com/fwlink/?LinkID=113216)


Examples:

$AuthenticationMethodsJSON = Get-VPASAuthenticationMethods


Outputs:

JSON Object (AuthenticationMethods) if successful

$false if failed
